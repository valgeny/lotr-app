# LOTR Demo App

Demo App for the Lotr SDK

## Prerequisites
- NodeJs > 18 (Fetch module)

See also: 
- https://www.freecodecamp.org/news/node-version-manager-nvm-install-guide/
- https://dev.to/cloudx/nodejs-18-fetch-api-test-runner-module-and-more-2ckg

## Install
```
npm install
```


# Start
```
npm start
```
